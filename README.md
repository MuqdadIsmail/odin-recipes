# odin-recipes# Recipes Website

This project is a simple **Recipes Website** built using HTML. It features a homepage that serves as the main navigation hub, allowing users to browse different recipes.

## Features

* A homepage containing an **unordered list** of recipe links.
* Individual recipe pages for each dish.
* Each recipe page includes:

  * The name of the dish.
  * An image of the dish.
  * An unordered list containing the recipe ingredients or steps.
* Easy navigation between the homepage and recipe pages.

## Purpose

The goal of this project is to practice the fundamentals of HTML, including:

* Creating multiple web pages.
* Using hyperlinks for navigation.
* Working with unordered lists.
* Adding and displaying images.
* Structuring web content using semantic HTML elements.

## Technologies Used

* HTML5
